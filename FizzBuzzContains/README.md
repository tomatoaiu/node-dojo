# FizzBuzzContains

## rule
- print from 1 to 100  
- multiples of 3 print "Fizz"  
- multiples of 5 print "Buzz"  
- multiples of 15 print "FizzBuzz"  
- contains 3 print "Fizz"  
- contains 5 print "Buzz"  
- contains 3 and 5 print "FizzBuzz"
